# VitalSmile Plataforma Médica Integral

Este proyecto es una **demo funcional exclusiva** desarrollada desde cero para el sector salud. No es una plantilla genérica; es un prototipo diseñado para ofrecer una experiencia profesional, clara y sensible, ideal para clínicas que buscan optimizar la confianza del paciente y la gestión de turnos.

## Características Principales

* **Calendario Interactivo en Tiempo Real**: Sistema de gestión de turnos que permite navegar por meses, bloqueando fechas pasadas y días no laborales (domingos).
* **Validación de Datos Segura**: Los formularios de contacto incluyen lógica de validación para evitar errores de llenado y garantizar datos limpios.
* **Diseño Profesional y Responsivo**: Interfaz optimizada para dispositivos móviles, utilizando una paleta de colores que transmite calma y profesionalismo médico.
* **Gestión de Especialistas**: Sección dinámica para presentar al equipo médico con biografías detalladas mediante modales interactivos.
* **Filtros Inteligentes de Horarios**: El sistema detecta la hora actual y oculta turnos que ya han pasado para el día vigente.

## Tecnologías Utilizadas

Este desarrollo demuestra mi capacidad para manejar stacks modernos y escalables:

* **Next.js 15+** (App Router)
* **React** (Hooks y gestión de estados complejos)
* **TypeScript** (Código robusto y tipado)
* **Tailwind CSS** (Diseño moderno y adaptable)

## Instalación y Uso

Seguí estos pasos para ejecutar el proyecto en tu entorno local:

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/mateoareosa/vitalsmile.git](https://github.com/mateoareosa/vitalsmile.git)
   npm install
   npm run dev
   ```
   
   Abre http://localhost:3000 en tu navegador para ver el resultado.
